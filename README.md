# Raqmy Platform

مشروع منصة بيع متعددة التجار (Multi-Vendor E-commerce Platform) تم تطويره باستخدام Bolt كنسخة أولية MVP.

## مميزات المشروع:
- إنشاء متجر للتاجر
- إضافة المنتجات وإدارتها
- سلة المشتريات
- الدفع الإلكتروني (Paymob / Stripe)
- لوحة تحكم للتاجر
- لوحة تحكم للمدير
- نظام عمولة
- نظام الاشتراكات

## الهدف:
بناء منصة تجارة إلكترونية سعودية قابلة للتطوير وإطلاقها كخدمة SaaS.

## حالة المشروع:
قيد التطوير الفعّال. يتم تحديث هذا المستودع مع تقدم مراحل المشروع.
